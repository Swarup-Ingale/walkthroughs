# URL
https://hackmyvm.eu/challenges/challenge.php?c=029

# Concept
To find all the domains of any domain name using various tools (here subfinder)

# Method of Solve

- Go to the terminal and perform subfinder for domain names using -d flag for hackmyvm.eu domain name
