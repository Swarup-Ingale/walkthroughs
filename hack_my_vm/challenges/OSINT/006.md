# URL
https://hackmyvm.eu/challenges/challenge.php?c=006

# Concept
How to perform NSLookUp on any website to get information about it.

# Method of Solve
- Go to the Challenge website and read the description carefully
- copy the url and perform nslookup on it with the -q flag set to TXT because it is mentioned in TXT in the challenge
  ```
    └─$ nslookup -q=TXT hackmyvm.eu
        Server:		10.0.2.3
        Address:	10.0.2.3#53
        
        Non-authoritative answer:
        hackmyvm.eu	text = "hmv{sasviyalin}"
        
        Authoritative answers can be found from:
  ```
- The Flag is **hmv{sasviyalin}**
