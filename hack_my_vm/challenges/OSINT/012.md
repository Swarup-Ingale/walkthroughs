# Concept
How can we perform a Audio file to Text transcription using open source platforms.

# Method of Solve
- Go to the Challange Website and download the audio file
- Upload the audio file on a Open source website that can transcribe audio into text such as "clipto AI"
- Submit the transcribed text as flag
- The flag is **HMV{ilovelanguages}**
