# Concept 
Analyze pcap file in wireshark and understand how useful it can be in network traffic analysis

# Method of Solve
- get the pcap file from the challenge link
- open it in wireshark and look for the TCP packets transmitted and ack recieved all the frames
- then look for the one with hint for flag
- it will say **flag{source_mac_address/data_length}**
- then look for the source mac address and data lenght of that same packet
- that is the final flag
- The flag is **flag{26:0e:ad:ef:c6:65/63}**
